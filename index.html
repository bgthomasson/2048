How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

Also, you can use H for left, J for down, K for up, and L for right, which I discovered by accident!

This is a fork to test using new CSS features like flexbox (97% support in 2017) instead of floats. Also, just for clarity in coding, some HTML5 semantic elements (95% support in 2017).

Created by Gabriele Cirulli. Based on 1024 by Veewo Studio and conceptually similar to Threes by Asher Vollmer.
